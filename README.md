# ship monorepo

Monorepo for the ship ecosystem.

## Packages

- `@i-santos/ship`: CLI orchestrator (PR/release/setup + adapter runtime)
- `@i-santos/agent`: deterministic task lifecycle FSM engine

## Workspace commands

```bash
npm run test
npm run check
npm run changeset
npm run release
```
